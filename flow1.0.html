Visual Flowchart

Google Drive Document Upload & Management (FLOW 1.0)

1. Initiate Upload

User clicks the "+" button (for new card/backfilling) or "Upload" on an existing card.

2. Categorize

System presents nested selection:
Level 1: Document Type (Ex: Bid Request).
Level 2 (if Bid Request): Trade (Ex: Gutter).

3. Organize in Drive

System dynamically identifies/creates folder structure in Google Drive: `[Job]/Original Bids/[Trade]/` (includes `Archived` subfolder).

4. Upload File

User selects file from device. System saves file to the correct Drive folder. Success feedback is displayed.

5. View in FLOW

File appears listed in the corresponding FLOW App card, with details (name, date) and status indication (active/archived).

6. Manage Files

In FLOW App:
- Archive: Moves file to `Archived` folder in Drive.
- Delete: Removes file from Drive (with confirmation).
- View/Download: Deep link to file in Drive.

7. Process Complete

Document uploaded, organized, and manageable through FLOW, synchronized with Google Drive.
